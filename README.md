# 📊 Интерактивный дашборд аналитики продаж РФ

## 📝 Описание проекта
Данный проект представляет собой функциональный интерактивный веб-дашборд для автоматизации анализа ключевых показателей эффективности (KPI) отдела продаж за последний квартал. 

Инструмент разработан в соответствии с техническим заданием для менеджеров и руководителей департамента продаж. Он агрегирует транзакционные данные, сопоставляет их со справочниками сотрудников и плановыми показателями, а также выводит аналитику на интерактивную карту России.

## 🚀 Основной функционал
* **Очистка данных и обработка аномалий:** Встроенная логика фильтрации отмененных заказов для исключения искажения итоговых KPI.
* **Интерактивные срезы (Фильтры):** Динамическая фильтрация всех графиков на лету по *Регионам (Федеральным округам)* и *Категориям товаров*.
* **Географический анализ:** Интерактивная карта РФ (Choropleth map) с автоматической фокусировкой и цветовым кодированием объема чистой прибыли.
* **План-факт анализ:** Визуальное сравнение целевых показателей выручки сотрудников с реальным фактом продаж с выводом грейдов (`Senior`/`Middle`/`Junior`).
* **Временные тренды:** Линейный график динамики продаж по датам для отслеживания спадов и пиков активности клиентов.

## 🛠️ Архитектура данных (Архитектура «Звезда»)
Проект полностью автономен и не требует внешних подключений к базам данных. Внутри кода объединены 4 таблицы:
1. **Таблица транзакций (Факт):** Поля заказов, даты, цен, скидок и стоимости доставки.
2. **Таблица планов (Цели):** Месячные таргеты по выручке и чистой прибыли для менеджеров.
3. **Справочник сотрудников:** База данных менеджеров с привязкой к отделам и грейдам.
4. **Справочник календаря:** Автоматически генерируемый в памяти слой временных интервалов.

## 💻 Локальный запуск (Альтернативный вариант)
Если вы хотите запустить проект локально на своем компьютере, выполните следующие шаги:

1. Клонируйте репозиторий или скачайте файлы.
2. Установите необходимые библиотеки:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите дашборд через терминал:
   ```bash
   streamlit run app.py
   ```

## 🧰 Технологический стек
* **Python** (Pandas) — предобработка данных, бизнес-логика, JOIN-объединение таблиц.
* **Streamlit** — фреймворк для создания веб-интерфейса дашборда.
* **Plotly** — интерактивные графики и работа со слоями GeoJSON для карты России.
